# Home Assistant Setup

This repository contains my personal Home Assistant configuration, custom cards, and components.

## Structure

- `custom_cards/` - Custom Lovelace cards
- `custom_components/` - Custom Home Assistant components  
- `configuration/` - Home Assistant configuration files
- `automations/` - Automation YAML files
- `scripts/` - Home Assistant scripts
- `themes/` - Custom themes
- `docs/` - Documentation and notes
